TypeScript

Mi is az a TypeScript?

  • JavaScript szuperhalmaza (a JS kód valid TS kód)
  • Modern típusos JavaScript
  • támogatja az új ECMAScript funkciókat => régebbi verzióra compile-ol
  • fájlok kiterjesztése .ts
  • IDE támogatás
  • TSLint

Compile-olt kód


                  class User {
                    private name: string;
                    constructor(name: string) {
                      this.name = name;
                    }
                    public greet() {
                      return "Hi, " + this.name;
                    }
                  }

                  const user = new User("Peter");
                  console.log(user.greet());
                

                  var User = (function () {
                    function User(name) {
                      this.name = name;
                    }
                    User.prototype.greet = function () {
                      return "Hello, " + this.name;
                    };
                    return User;
                  }());

                  var user = new User("Peter");
                  console.log(user.greet());

Angular Services


                export class HeroService {
                  private heroes: Hero[] = [];

                  constructor(
                    private backend: BackendService,
                    private logger: Logger) { }

                  getHeroes() {
                    this.backend.getAll(Hero).then( (heroes: Hero[]) => {
                      this.logger.log(`Fetched ${heroes.length} heroes.`);
                      this.heroes.push(...heroes); // fill cache
                    });
                    return this.heroes;
                  }
                }
